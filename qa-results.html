3. 수동 테스트 필요 항목 (자동화 불가)

아래 항목은 브라우저에서 직접 확인이 필요합니다. temp01.zpack.app 하나에서 테스트하면 모든 템플릿에 적용됩니다 (서브페이지/기능 공통).

어드민 로그인 · Visual Editor

  • 올바른 비밀번호 로그인 → Edit Mode 버튼 헤더 표시 확인
  • 잘못된 비밀번호 → 오류 메시지 표시 확인
  • 로그인 후 새로고침 → 세션 유지 확인
  • Edit Mode 진입 → 편집 가능 요소 하이라이트 표시 확인
  • 텍스트 클릭 편집 → 저장 → 새로고침 후 유지 확인
  • 이미지 클릭 → 업로드 → 저장 → 반영 확인
  • 테마 컬러 변경 → 저장 → 전체 반영 확인
  • 회사명 변경 → 헤더·푸터 즉시 반영 확인
  • Edit Mode 종료 → 편집 UI 비활성화 확인
  • 로그아웃 → Edit Mode 버튼 사라짐 확인

게시판 CRUD (어드민 대시보드)

  • 공지사항 새 글 작성 → 발행 → /notice 목록에 표시 확인
  • 글 상세 클릭 → 상세 페이지 이동 확인
  • 글 수정 → 저장 → 상세 페이지 반영 확인
  • 글 삭제 → 확인 다이얼로그 → 삭제 후 목록 제거 확인
  • 리치텍스트 에디터: Bold, Italic, 이미지 삽입, 링크 동작 확인
  • FAQ 항목 추가 → 발행 → 아코디언 동작 확인
  • 포트폴리오 아이템 이미지 업로드 → /portfolio 갤러리 반영 확인
  • 어드민 비밀번호 변경 → 새 비밀번호 로그인 성공 확인
