이 가이드를 따라하면 내 웹사이트가 인터넷에 올라갑니다.

1
GitHub 계정 만들기
GitHub는 내 웹사이트 파일을 보관하는 창고입니다. 무료로 사용할 수 있어요.
1-1
GitHub 사이트 접속

아래 버튼을 클릭하거나, 브라우저에서 github.com을 입력해 접속하세요.

🔗 GitHub 가입 페이지 열기
💡 이미 GitHub 계정이 있다면 이 단계는 건너뛰세요.
1-2
이메일 · 비밀번호 · 사용자명 입력

① 자주 사용하는 이메일 주소 입력
② 비밀번호 설정 (영문+숫자 조합 8자 이상)
③ 사용자명(username) 입력 — 영문·숫자·하이픈만 가능

⚠️ 사용자명은 나중에 웹사이트 주소에도 들어가니 기억하기 쉬운 이름으로 정하세요.
1-3
이메일 인증

가입 후 입력한 이메일로 인증 메일이 옵니다.
메일을 열고 "Verify email address" 버튼을 클릭하세요.

✅ 인증이 완료되면 GitHub 가입 끝!
2
ZeroPack에서 GitHub 연결하기
ZeroPack이 내 GitHub에 저장소를 자동으로 만들어줍니다.
2-1
ZeroPack 로그인 후 대시보드 접속

ZeroPack(zpack.app)에 로그인하고 대시보드로 이동하세요.

🔗 zpack.app 열기
2-2
"GitHub 연결" 버튼 클릭

대시보드에서 "GitHub 연결" 또는 "Connect GitHub" 버튼을 클릭하세요.

GitHub 로그인 화면이 뜨면 방금 만든 계정으로 로그인합니다.

ℹ️ "ZeroPack이 내 GitHub 계정에 접근하도록 허용할까요?" 라는 화면이 뜨면 Authorize(허용)를 클릭하세요. 안전한 과정입니다.
2-3
저장소 자동 생성 확인

ZeroPack이 내 GitHub에 저장소(Repository)를 자동으로 만듭니다.
GitHub 사이트에서 확인하려면:

github.com → 우측 상단 프로필 → Your repositories

✅ 저장소가 보이면 연결 성공!
3
Vercel 계정 만들기
Vercel은 내 웹사이트를 인터넷에 올려주는 서비스입니다. 기본 사용은 무료예요.
3-1
Vercel 사이트 접속

아래 버튼을 클릭해 Vercel 가입 페이지로 이동하세요.

🔗 Vercel 가입 페이지 열기
3-2
"Continue with GitHub"로 가입 (추천)

가입 화면에서 "Continue with GitHub" 버튼을 클릭하세요.
GitHub 계정으로 바로 가입되어 따로 비밀번호를 만들 필요가 없습니다.

💡 이 방법이 가장 간편하고, 나중에 자동 배포 연결도 쉬워집니다.
3-3
권한 허용 및 가입 완료

"Vercel이 내 GitHub에 접근하도록 허용할까요?" 화면이 뜨면
Authorize Vercel을 클릭하세요.

이름·직업 등 몇 가지 질문이 나올 수 있습니다. 아무렇게나 입력해도 됩니다.

✅ Vercel 대시보드 화면이 보이면 가입 완료!
4
ZeroPack에서 배포하기
이제 내 웹사이트를 인터넷에 올릴 차례입니다. 버튼 몇 번이면 끝!
4-1
ZeroPack에서 "Vercel 연결" 클릭

ZeroPack 대시보드로 돌아와서 "Vercel 연결" 또는 "Deploy" 버튼을 클릭하세요.

Vercel 로그인 화면이 뜨면 방금 만든 계정으로 로그인합니다.

4-2
프로젝트 가져오기(Import)

Vercel이 GitHub 저장소 목록을 보여줍니다.
ZeroPack이 만들어 준 저장소 옆 Import 버튼을 클릭하세요.

ℹ️ 저장소가 보이지 않으면 "Adjust GitHub App Permissions"를 클릭해 권한을 추가하세요.
4-3
"Deploy" 버튼 클릭

설정 화면이 나오면 아무것도 바꾸지 않고 바로 Deploy를 클릭하세요.
ZeroPack이 이미 최적 설정을 해뒀습니다.

배포가 진행되는 동안 로딩 화면이 보입니다. 보통 1~2분 소요됩니다.

🎉 완료되면 내 웹사이트 주소가 나타납니다!
4-4
배포 주소 확인

Vercel이 자동으로 주소를 생성합니다. 예시:

https://내-사이트이름.vercel.app

이 주소를 브라우저에 입력하면 내 웹사이트가 열립니다. 🎊

💡 ZeroPack 대시보드에서 배포 주소를 입력하면 이후 내용 수정 시 자동으로 반영됩니다.
🎊
초기 설정 완료!
GitHub와 Vercel 연결이 끝났습니다.
이제 ZeroPack에서 내용을 수정하면 자동으로 웹사이트에 반영됩니다.
🏠 ZeroPack 대시보드 ▲ Vercel 대시보드 🐙 GitHub
?
자주 묻는 질문
막히는 부분이 있다면 아래를 확인하세요.
GitHub가 뭔가요?
웹사이트 파일을 저장하는 온라인 창고입니다. 수정 내역도 자동으로 기록되어 언제든 이전 버전으로 돌아갈 수 있습니다.
Vercel이 뭔가요?
GitHub에 저장된 파일을 읽어서 실제 인터넷 주소로 서비스해주는 플랫폼입니다. GitHub에 변경사항이 올라오면 자동으로 웹사이트를 업데이트합니다.
비용이 드나요?
GitHub와 Vercel 모두 개인 무료 플랜으로 ZeroPack을 충분히 사용할 수 있습니다. 트래픽이 매우 많아지면 유료 플랜이 필요할 수 있지만, 초기에는 걱정하지 않아도 됩니다.
저장소(Import)가 Vercel 목록에 안 보여요.
Vercel의 GitHub App 권한 설정에서 해당 저장소를 추가해야 합니다. "Configure GitHub App" 또는 "Adjust Permissions" 링크를 클릭해 저장소를 선택하세요.
배포 후 내용을 바꾸면 어떻게 되나요?
ZeroPack에서 수정 → 저장하면 GitHub에 자동 업로드됩니다. Vercel이 이를 감지하고 자동으로 웹사이트를 업데이트합니다. 별도 작업 없이 1~2분 내에 반영됩니다.
