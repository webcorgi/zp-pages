02 — Description (PH 본문)
ZeroPack is a landing page builder with a radical idea: you own everything.

One payment. No monthly fees. Your code lives in your GitHub. Your site runs on your Vercel. Even if ZeroPack disappears tomorrow, your website keeps running — forever.

How it works:
1. Purchase a license
2. Go to zpack.app and enter your license key
3. Connect GitHub + Vercel (guided setup)
4. Your site deploys automatically in ~3 minutes
5. Edit text, images, and colors with our Visual Editor — no code needed

Why we built this:
Landing page tools charge $30-50/month. That's $600/year for a simple website you don't even own. We think that's broken. ZeroPack flips the model: pay once, own it forever, host it free.

Key features:
✓ 3-minute auto-deploy via GitHub + Vercel
✓ Visual Editor — click to edit anything
✓ 100% code ownership (your GitHub repo)
✓ Zero hosting costs (Vercel free tier)
✓ 15 languages supported
✓ Mobile-optimized out of the box
✓ 10 professional templates included
03 — 첫 댓글 (Maker Comment) — 런칭 당일 직접 작성
Hi Product Hunt! 👋

I'm Daehyun, a solo developer from South Korea.

I built ZeroPack because I was frustrated paying $40/month for a landing page tool that "owned" my website. When I stopped paying, my site was gone. That felt wrong.

So I built the opposite:
- You pay once → you get a license key
- Enter it on zpack.app → it forks the template to YOUR GitHub, creates a Vercel project under YOUR account
- 3 minutes later, your site is live at yourdomain.zpack.app
- Edit everything visually. No code needed.
- Stop paying? Your site keeps running. Forever.

The hardest part was building the zero-config setup pipeline — automatically wiring GitHub, Vercel, environment variables, and subdomain in one click without any user touching a terminal.

Would love your feedback! Happy to answer any questions 🙏
