## How I Built ZeroPack: A One-Time-Payment Website Builder for Small Businesses ### The frustration that started it all I kept seeing the same pattern: a local clinic, a language academy, a small design studio — all needing a basic business website, all getting quoted $2,500–5,000 by developers, or pushed toward $100–300/month SaaS subscriptions they'd need forever. The math never made sense. A 5-person business paying $150/month for a website is spending $1,800/year on something that should be a one-time cost. ### The insight GitHub + Vercel offer genuinely free hosting for static sites and lightweight apps. If I could build a product that deploys directly into a customer's GitHub repo and Vercel account, the hosting cost would be zero — permanently. ### Tech stack - **Frontend:** Next.js (static generation for speed + free Vercel hosting) - **Backend:** Lightweight API routes for admin functionality - **Database:** None for static content; simple JSON/markdown for CMS content - **Deployment:** Vercel auto-deploy from GitHub push - **Admin panel:** Custom-built with Visual Editor ### The business model One-time payment ($150 beta / $225 regular). No subscriptions, no upsells, no hidden fees. The customer gets the full GitHub repo — they own the code completely. Revenue model is simple: charge once, deliver value forever. The bet is that word-of-mouth from happy small business owners compounds over time. ### What's live today - Multi-board system (notice, blog, FAQ, job listings) - Portfolio gallery - Contact form with routing - Admin dashboard + Visual Editor - Automated GitHub + Vercel deployment ### Biggest challenge Convincing small business owners that "you own the code" is actually a feature, not a scary technical detail. Working on better onboarding copy for this. ### Revenue so far Early beta stage. First customers onboarding now. ### What's next - Polished onboarding flow - More templates - Product Hunt launch - English market expansion

"GitHub이 뭔지도 모르는데 제가 할 수 있을까요?" 이 세 가지 문제를 동시에 해결하는 제품이 없다는 걸 확인하고, ZeroPack을 만들었습니다. ### 핵심 아이디어 GitHub + Vercel은 둘 다 무료 플랜이 있습니다. 이 두 플랫폼을 연결해서 고객의 계정에 홈페이지를 자동 배포하면, 이후 서버 비용이 0원이 됩니다. 고객은 한 번 결제하고, 코드는 자기 GitHub에 올라가고, 사이트는 Vercel에 무료로 호스팅됩니다. 영원히. ### 실제 만들어진 것 - 게시판 4종 (공지/블로그/FAQ/채용) - 포트폴리오 갤러리 - 문의 폼 - 관리자 로그인 + 대시보드 - Visual Editor (코딩 없이 편집) ### 가격 베타 특가 ₩200,000 — 정가는 ₩300,000입니다. 데모 사이트 한번 구경해 보세요. 관리자로 직접 로그인해서 글도 써볼 수 있습니다. 👉 temp01.zpack.app
