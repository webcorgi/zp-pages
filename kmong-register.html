✅ 이런 분께 딱 맞아요 • 아임웹·카페24 월 구독료가 부담스러운 분 • 홈페이지 만들고 싶은데 코딩을 전혀 모르는 분 • 개발자에게 맡겼다가 수백만원 날린 경험이 있는 분 • 한 번 만들면 추가 비용 없이 영구 운영하고 싶은 분 --- 🚀 ZeroPack이 뭔가요? ZeroPack은 누구나 5분 안에 내 랜딩페이지를 배포할 수 있게 해주는 서비스예요. 라이선스 키 하나로 — • GitHub + Vercel 자동 연동 • 전문적인 랜딩페이지 즉시 배포 • Visual Editor로 클릭해서 내용 수정 • 15개 언어 자동 지원 --- 💰 비용 비교 아임웹 5년 비용: 월 33,000원 × 60개월 = 약 200만원 ZeroPack: 단 한 번 결제 → 이후 0원 5년간 약 180만원 절약됩니다. --- 📦 구매 후 진행 과정 1. 구매 완료 → 라이선스 키 즉시 발급 2. zpack.app 접속 → 키 입력 3. GitHub 계정 연결 (무료, 5분) 4. 배포 완료 → 내 사이트 URL 생성 --- ❓ 자주 묻는 질문 Q. 코딩을 전혀 모르는데 진짜 할 수 있나요? A. 네! 버튼 클릭만 할 줄 알면 됩니다. 가이드가 하나하나 알려드려요. Q. 정말 평생 무료인가요? A. 네! Vercel 무료 플랜으로 운영되기 때문에 호스팅 비용 0원, 구독료 0원입니다. Q. ZeroPack이 없어지면 내 사이트도 사라지나요? A. 아니요! 사이트는 내 GitHub + 내 Vercel에서 운영되므로 ZeroPack과 무관하게 영구 운영됩니다. Q. 내 도메인 연결도 되나요? A. 네! 도메인 연결 방법도 가이드에 포함되어 있어요. --- 🔥 지금 베타 출시 특가 진행 중 정가 300,000원 → 베타 특가 200,000원 (베타 기간 종료 후 정가로 복귀)

크몽 서비스 등록 체크리스트
1
판매자 인증

크몽 › 판매자 센터 › 전문가 인증 완료 (신분증 인증 필요)

2
서비스 등록 시작

판매자 센터 › 서비스 관리 › 새 서비스 등록

3
위 자료 순서대로 입력

카테고리 → 제목 → 한 줄 소개 → 상세 설명 → 패키지 → 태그

4
썸네일 업로드

최소 3장 이상 권장 (메인 1장 + 설명 2장)

5
검수 제출 → 승인 대기

보통 1~3 영업일 소요. 승인 후 즉시 판매 시작.
