2. 발견된 이슈

🔴 [치명] IP 언어 감지 오류 — zpack.app 접속 시 /zh(중국어) 리다이렉트
증상: zpack.app 접속 시 /ko 아닌 /zh(중국어)로 리다이렉트됨
영향: 한국 사용자가 중국어 페이지를 보게 됨. 크몽 구매 후 첫 진입 경험 훼손
원인 추정: IP 지오로케이션 서비스가 해당 IP를 중국으로 잘못 분류
조치: 먼저 본인 브라우저로 zpack.app 접속해서 직접 확인 필요. /zh 뜨면 미들웨어 fallback 언어를 en 또는 ko로 변경
🟡 [권고] OG 메타 설명 불일치 — temp01-06, temp01-10
temp01-06 (Seoul Hanbit Hospital): og:description = "Build and manage beautiful websites..." (병원에 부적절)
temp01-10 (Z-Tech Solutions): og:description = "Build and manage beautiful websites..." (불일치)
조치: content.json에서 siteDescription 수정 필요
🟡 [권고] /admin-login 수동 확인 필요
증상: /admin-login 200 응답 — URL은 공개되어 있음 (footer 하단에 매우 연한 색으로 표시)
확인: 실제 인증 없이 관리자 기능 접근되는지 수동 테스트 필요
🟡 [낮음] zpack.app 초기 응답 3.5초
원인: 언어 감지 + 리다이렉트 처리 지연 (템플릿들은 평균 1.2s)
조치: 언어 감지 이슈 해결 시 자연히 개선될 가능성 높음
